# VS Code Extensions Installer

Ce dépôt contient un script permettant d'installer automatiquement mes extensions VS Code préférées sous Linux.

## Fichiers

* `extensionVsCodeLinux.sh` : Linux

## Utilisation

### Linux

```bash
chmod +x extensionVsCodeLinux.sh
./extensionVsCodeLinux.sh
```

## Vérification

```bash
code --version
```
